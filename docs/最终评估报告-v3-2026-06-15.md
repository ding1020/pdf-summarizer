### 🟠 阶段二：支付自动化（Day 3-7）—— 商业化闭环

> **此阶段需要你手动完成 1 项操作：注册 Creem（~2小时）**

#### 步骤 2.1：注册 Creem（你必须做）

| 步骤 | 操作 |
|------|------|
| 1 | 访问 [creem.io](https://creem.io) → Sign Up → 个人开发者 → 邮箱注册 |
| 2 | Settings → Payouts → 绑定**支付宝账号** |
| 3 | Verification → 上传**身份证正反面** + 人脸识别 |
| 4 | 等待审核（通常几小时内通过） |
| 5 | Products → Create Product |
|  | · Pro Monthly: USD 7.99 / EUR 7.49 / GBP 6.49 / JPY 980 / CNY 49（可选） |
|  | · Pro Yearly: USD 69 / EUR 64 / GBP 55 / JPY 8400 / CNY 399 |
|  | · 类型选 Recurring，记录每个价格的 **Price ID** |
| 6 | Developers → API Keys → 生成 Secret Key |
| 7 | Webhooks → 添加 `https://www.pdfsum.com/api/webhooks/creem` |
|  | · 选择事件: checkout.session.completed / subscription.updated / subscription.deleted |
|  | · 复制 Webhook Secret |
| 8 | 环境变量（Vercel Dashboard → Settings → Environment Variables）： |

```bash
CREEM_SECRET_KEY=
CREEM_WEBHOOK_SECRET=
NEXT_PUBLIC_PRICE_MONTHLY_USD=
NEXT_PUBLIC_PRICE_YEARLY_USD=
RESEND_API_KEY=
EMAIL_FROM=noreply@pdfsum.com
```
